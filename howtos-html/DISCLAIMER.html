Disclaimer

These HOWTOs are generated with AI assistance using the official Reltio documentation as source material. They're a starting point for understanding a feature — not a guarantee of accuracy, and not a replacement for the official Reltio documentation.

Two things to keep in mind.

Model non-determinism

AI models are probabilistic. Even with strict guardrails and a validation pass against the source documentation, generated content can misrepresent system behavior, flatten important edge cases, or introduce subtle inaccuracies. Verify anything you plan to act on in your own Reltio environment.

Content drift since the snapshot

Each guide shows the timestamp and topic count of the documentation snapshot it was generated from. The knowledge base behind these guides syncs twice a week — on Wednesdays and Fridays. Any documentation Reltio publishes between syncs isn't captured here, so at any given moment the content can lag the live docs.reltio.com by several days.

If a guide describes behavior that doesn't match your current tenant, docs.reltio.com is the source of truth.
